p Back To Top

  • align - ALIGNS LEFT, RIGHT, CENTER, JUSTIFIED (NOT SUPPORTED IN HTML5. USE CSS)

ul Back To Top

UNORDERED LIST TAG - MAKES CHILD LIST USE BULLET POINTS

  • compact - TELLS SYSTEM THAT THE LIST SHOULD RENDER SMALLER THAN NORMAL (NOT USABLE IN HTML5)
  • type - DISC/SQUARE/CIRCLE - CHANGES LIST MARKER (NOT USABLE IN HTML5)

<ul>This is the Title of My Unordered List</ul>

  • This is my Unordered List Item

ol Back To Top

ORDERED LIST TAG - MAKES CHILD LIST USE NUMBERED ITEMS

  • compact - SPECIFIES THAT LIST SHOULD RENDER SMALLER THAN NORMAL (NOT SUPPORTED IN HTML5)
  • reversed - SPECIFIES LIST NUMBERS TO GO IN REVERSE
  • start - NUMBER - SPECIFICES START VALUE OF THE LIST
  • type - 1/A/a/I/i - SPECIFIES THE MARKER TO USE

<ol>This is the Title of My Ordered List</ol>

  1. This is item number one

li Back To Top

LIST ITEM TAG - WILL USE BULLET POINTS OR NUMBERED ITEMS DEPENDING ON IF IT IS NESTED WITHIN AN ORDERED LIST OR UNORDERED LIST TAG

  • type - 1/A/a/I/i/disc/square/circle - SPECIFIES WHICH KIND OF BULLET POINT WILL BE USED (NOT USABLE IN HTML5)
  • value - NUMBER - SPECIFIES VALUE OF A LIST ITEM. FOLLOWING ITEMS WILL INCREMENT FROM THAT NUMBER (ol LISTS ONLY)

<li>Apples</li>

a Back To Top

ANCHOR - DEFINES A HYPERLINK

  • download - filename - SPECIFIES THE TARGET WILL BE DOWNLOADED WHEN USER CLICKS ON IT
  • href - URL - SPECIFIES PAGE WHERE THE LINK GOES
  • hreflang - language_code - SPECIFIES LANGUAGE IN THE DOCUMENT
  • ping - list_of_urls - SPECIFIES LIST OF URL's
  • rel - alternate/author/bookmark/external/help/license/next/nofollow/noreferrer/noopener/prev/search/tag - Specifies the relationship between the current document and the linked document
  • target - _blank/_parent/_self/_top/framename - Specifies where to open the linked document
  • type - media_type - Specifies the media type of the linked document

<a href="https://www.facebook.com">Go to facebook.com</a>

img Back To Top

IMAGE TAG - DEFINES AN IMAGE IN AN HTML PAGE

  • src - URL - FINDS SOURCE OF IMAGE
  • alt - TEXT - SPECIFIES THE ALTERNATE TEXT
  • height - PIXELS
  • width - PIXELS

<img src="https://www.facebook.com/image" alt="This is an image">

head Back To Top

link Back To Top

LINK TAG - DEFINES LINK BETWEEN A DOCUMENT AND AN EXTERNAL RESOURCE. GOES ONLY IN THE HEAD ELEMENT

  • crossorigin - ANONYMOUS/USE CREDENTIALS - SPECIFIES HOW ELEMENT HANDLES CROSS-ORIGIN REQUESTS
  • href - URL - SPECIFIES THE LOCATION OF DOCUMENT
  • hreflang - language_code - SPECIFIES LANGUAGE OF TEXT IN THE LINKED DOCUMENT
  • media - media_query - SPECIFIES ON WHAT DEVICE THE LINKED DOCUMENT WILL BE DISPLAYED
  • rel - alternate/author/dns-prefetch/help/icon/license/next/pingback/preconnect/prefetch/preload/prerender/prev/search/stylesheet - REQUIRED. SPECIFIES RELATIONSHIP BETWEEN CURRENT DOCUMENT AND LINKED DOCUMENT.
  • sizes - HEIGHT/WIDTH - SPECIFIES THE SIZE OF THE LINKED RESOURCE. ONLY FOR rel="icon"
  • type - media_type - SPECIFIES MEDIA TYPE OF LINKED DOCUMENT

<link rel="stylesheet" type="text/css">

title Back To Top

BODY TAG - CONTAINS CONTENT OF HTML DOCUMENT - TEXT, HYPERLINKS, IMAGES, TABLES, ETC.

  • attributes not supported in HTML5

span Back To Top

SPAN TAG - USED TO GROUP INLINE ELEMENTS IN A DOCUMENT

<span><body>Body Text</body></span>

div Back To Top

DEVISION TAG - DEFINES SECTION IN AN HTML DOCUMENT

  • class - HELPS WITH CSS STYLING - CLASS TAGS CAN BE REPEATED
  • id - HELPS WITH CSS STYLING/SEPARATION - TAGS SHOULD NOT BE REPEATED

<div id="top">Text Here</div>

header Back To Top

HEADER TAG - CONTAINS INTRO CONTENT / NAVIGATION LINKS. TYPICALLY HAS 1+ HEADING ELEMENTS, LOGO OR ICON, AUTHORSHIP INFO

<header>

<h1>Information</h1>

</header>

footer Back To Top

DEFINES FOOTER SECTION OF THE PAGE. TYPICALLY CONTAINS CONTACT INFO, COPYRIGHT INFO, BACK TO TOP LINKS, ETC.

<footer>Contact Us: <a href="mailto:someone@gmail.com"</footer>

main Back To Top

MAIN TAG - SPECIFIES MAIN CONTENT IN A DOCUMENT. CONTENT SHOULD BE UNIQUE TO THE DOCUMENT AND NOT CONTAIN CONTENT REPEATED LIKE SIDEBARS, NAV LINKS, COPYRIGHT INFO, SITE LOGOS, AND SEARCH FORMS.

<main>

<h1>Heading</h1>

</main>

section Back To Top

SECTION TAG - DEFINES SECTIONS IN A DOCUMENT, LIKE CHAPTERS, HEADERS, ETC.

  • class - HELPS WITH CSS STYLING - CLASS TAGS CAN BE REPEATED
  • id - HELPS WITH CSS STYLING/SEPARATION - TAGS SHOULD NOT BE REPEATED

<section>This Is A Section</section>

<section class="info">This Is A Section</section>

aside Back To Top

ASIDE TAG - SEPARATES CONTENT (USUALLY INTO SIDEBARS OR CALL-OUTS) BUT IS STILL RELATED TO THE MAIN CONTENT

<aside>

<p>This info is contained within a sidebar</p>

</aside>
